Weather app

API-driven weather app, built using React. It provides accurate 5-day forecast & historical data for any global location. This data includes six elements, such as current date & time, wind, humidity, cloudiness, and temperature. Additionally, it has a function of weather conversion.

Dictionary app

Responsive online English dictionary app with an interactive background, created by using React and API. This app includes features such as search engine, audio pronunciation, example sentence of the word and synonyms.
